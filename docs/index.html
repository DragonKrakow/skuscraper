EAN Code Data Scraper

This project scrapes and aggregates product data by EAN code for Italy (IT) and Poland (PL) using free/open sources.

Unified Output Schema

{
  "ean": "8000500310427",
  "product_name": "Example Product",
  "brand": "Example Brand",
  "category": "example-category",
  "price": 12.99,
  "currency": "EUR",
  "source": "open_food_facts,trovaprezzi",
  "market": "IT",
  "scraped_at": "2026-04-15T17:00:00+00:00"
}

CLI Examples

python italy/scraper_it.py --ean 8000500310427
python poland/scraper_pl.py --batch /absolute/path/eans.txt --csv data/results_pl.csv
