Product Search (Italy + Poland)

Enter EAN or product name, then run the GitHub Actions workflow manually to generate a downloadable CSV artifact.

New — Hertwill batch mode: select mode = hertwill in the workflow and set csv_path to a repo-relative path (e.g. inputs/hertwill/hertwill_skincare_bestsellers.csv) to scrape all products in that CSV at once.

Tip: Pages cannot run scrapers directly. Use this form to prepare inputs, then run the workflow from the Actions tab.

After the workflow completes, open the run page and download the CSV from the Artifacts section.
